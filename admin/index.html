Información sobre notificaciones

  • Confirmación: Se envía automáticamente al crear o confirmar una cita
  • Recordatorios: Se procesan cada hora mediante un cron job
  • Cancelación: Se envía automáticamente al cancelar una cita
  • Reprogramación: Se envía cuando se cambia la fecha u hora de una cita
  • Agradecimiento: Se envía al marcar una cita como completada
